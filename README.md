# Show All Links — Extension Chrome/Edge

Cette extension affiche tous les liens présents sur la page active et les regroupe en liens internes et externes.

Installation pour développement :

1. Ouvrir `chrome://extensions/` ou `edge://extensions/`.
2. Activer "Mode développeur".
3. Cliquer sur "Charger l'extension non empaquetée" et sélectionner le dossier du projet.
4. Ouvrir une page web et cliquer sur l'icône de l'extension pour voir les liens.
